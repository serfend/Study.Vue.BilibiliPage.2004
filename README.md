# bilibili.com implement base on vue

## get a start

```
npm install
npm run serve // start project
npm run build // pack project

```

## DevVia

- Vue-Cli
- Vue
- Vue-Router
- Vuex
- ES 6 / 7 
- Sass
- Axios
